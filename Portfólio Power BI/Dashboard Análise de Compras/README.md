Desenvolvimento de painel interativo de Business Intelligence (BI) para a área de Compras/Suprimentos. O objetivo deste projeto foi transformar planilhas de dados brutos em uma ferramenta visual para monitoramento de KPIs e tomada de decisão estratégica.

Principais funcionalidades implementadas:

Monitoramento de KPIs: Cartões dinâmicos para Total Gasto, Descontos Obtidos e Produto Curva A.

Análise Temporal: Acompanhamento mensal do volume de compras (Preço Original vs. Com Desconto).

Gestão de Fornecedores: Ranking de fornecedores por volume financeiro para apoiar negociações.

Geolocalização: Mapa interativo para análise de distribuição de compras por região.

Top 10 Produtos: Visualização clara dos itens de maior saída/impacto no orçamento.
